Commandes ssh

Télécharger la clé ssh gatling-gar.pem.

Connexion sur le controller (remplacer l'étoile par votre id):
$ ssh -i gatling-gar.pem ec2-user@controller-*.aws.xebiatechevent.info

Connexion sur la webapp (remplacer l'étoile par votre id):
$ ssh -i gatling-gar.pem ec2-user@controller-*.aws.xebiatechevent.info

Cloner le repo de sources en local (remplacer l'étoile par votre id):
$ ssh-agent
$ ssh-add /your/path/gatling-gar.pem
$ git clone ssh://ec2-user@controller-3.aws.xebiatechevent.info/home/ec2-user/repos/tinyapp
